생활코딩

웹프로그래밍 공부를 위한 임시 페이지 입니다. 위의 생활코딩 홈페이지에 들어가시면 자세한 사항 확인하실 수 있습니다. localhost를 실행시키려면 wampstack manager프로그램을 실행시켜 appache서버를 restart해야합니다. 구체적인 페이지를 로드하는 방법으로는 chrome에서 ctrl + o 를 누르거나 github 홈페이지에서 직접 실행시키면 됩니다.
